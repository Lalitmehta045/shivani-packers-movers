# 🚚 Shivani Packers & Movers

A modern, responsive, and professional web application for **Shivani Packers & Movers**, Bhopal's most trusted logistics partner. This project serves as the digital front door for the business, highlighting their comprehensive shifting services, transparent pricing, and robust customer testimonials.

## ✨ Key Features

- **Modern UI/UX**: Clean, professional, and accessible design optimized for conversion.
- **Fully Responsive**: Flawless experience across desktops, tablets, and mobile devices.
- **Dynamic Service Pages**: Individual, detailed pages for every unique service offered.
- **Optimized Performance**: Built with Vite and React for lightning-fast page loads.
- **SEO Ready**: Semantic HTML and structured data implemented for better search visibility.

## 📦 Services Offered

- **Local Home Shifting**: Safe, hassle-free household moves within Bhopal.
- **Office Relocation**: Efficient moving with minimal business downtime.
- **Intercity Shifting**: Reliable, long-distance moving across India.
- **Car & Bike Transport**: Secure vehicle transport in specialized carriers.
- **Packing & Unpacking**: Premium packing techniques and high-quality materials.
- **Warehouse & Storage**: Safe, climate-controlled storage solutions.

## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://react.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: React Router DOM
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: Motion (framer-motion)

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
- [Node.js](https://nodejs.org/) (v16.x or newer recommended)
- `npm` or `yarn`

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Lalitmehta045/shivani-packers.git
   cd shivani-packers
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the app:**
   Open [http://localhost:5173](http://localhost:5173) in your browser.
